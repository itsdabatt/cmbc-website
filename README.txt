CMBC Website Update Instructions

FILES:
- index.html = words and sections
- style.css = colors/design
- images/logo.png = church logo
- images/church.png = main church photo
- images/vbs.png = VBS banner

HOW TO UPDATE GITHUB:
1. Open your GitHub cmbc-website repository.
2. Click Add file > Upload files.
3. Drag in the new index.html, style.css, and images folder from this package.
4. When GitHub asks about replacing files, allow it.
5. Scroll down and click Commit changes.
6. Cloudflare Pages should update automatically.

EASY CHANGES:
- To change service times, open index.html and search for Service Times.
- To change the VBS section, search for Vacation Bible School.
- To change the photo, replace images/church.png with a new image using the same filename.
- To change the logo, replace images/logo.png with a new image using the same filename.
